Widget Hit Area Lab

Editor-only wrapper untuk widget kecil perlu dibesarkan, tapi konten aslinya jangan berubah.

Masalah utamanya ada di widget seperti Divider dan Button: border selection, hover target, dan rail toolbar terlalu kecil karena wrapper editor mengikuti dimensi konten mentah. Prototype ini membandingkan tiga arah supaya canvas tetap nyaman dipilih tanpa merusak parity frontend parser.

Rekomendasi Awal

Balanced Frame paling aman untuk builder kamu. Konten widget tetap apa adanya, tapi shell editor dibuat full-width dan diberi minimum height per tipe widget.
